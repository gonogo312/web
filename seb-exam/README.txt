ИНСТАЛАЦИЯ И НАСТРОЙКИ
================================================================================

ИЗИСКВАНИЯ:
- XAMPP с Apache + PHP 8.x + MySQL/MariaDB
- phpMyAdmin (включен в XAMPP)

СТЪПКИ ЗА ИНСТАЛАЦИЯ:

1. Копиране на проекта
   Копирайте цялата папка `seb-exam/` в XAMPP `htdocs/` директорията:
   C:\xampp\htdocs\seb-exam\

2. Създаване на база данни
   - Отворете phpMyAdmin: http://localhost/phpmyadmin
   - Кликнете "New" в лявата странична лента
   - Име на базата: `seb_exam`
   - Collation: `utf8mb4_unicode_ci`
   - Кликнете "Create"

3. Импортиране на схема
   - Изберете базата `seb_exam`
   - Кликнете таб "Import"
   - Изберете файл: `seb-exam/sql/schema.sql`
   - Кликнете "Go"

4. Конфигуриране на база данни (ако е необходимо)
   - Редактирайте `seb-exam/app/config/db.php`
   - Обновете `DB_USER`, `DB_PASS` ако MySQL credentials се различават от default
   - По подразбиране: DB_USER='root', DB_PASS='' (празна парола)

5. Стартиране на инсталатора
   - Отворете: http://localhost/seb-exam/public/install.php
   - Това ще вмъкне демо данни с правилни password hashes
   - ЗАПАЗЕТЕ API токена, който се показва на страницата за инсталация
   - ИЗТРИЙТЕ `public/install.php` след инсталация за сигурност!

6. Вход в системата
   - Отворете: http://localhost/seb-exam/public/login.php
   - Преподавател: `teacher1` / `teacher123`
   - Студент: `student1` / `student123` или `student2` / `student123`

================================================================================
СТРУКТУРА НА ПРОЕКТА
================================================================================

Проектът е уеб базирана система за управление на изпити с Safe Exam Browser 
конфигурации. Включва:

- Аутентификация с роли (Преподавател/Студент)
- CRUD операции за изпити и въпроси (MCQ, True/False, Short Answer)
- Escape Room редактор (node-based adventure games)
- SEB Config Generator (визуален wizard за генериране на Safe Exam Browser XML)
- Student Pass Prediction (sigmoid-based probability scoring)
- REST API с token authentication (за интеграция с Moodle)
- Security features (CSRF, rate limiting, session hardening, prepared statements)
